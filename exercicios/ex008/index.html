Principais Formatações

Negrito / Destaque

Temos um termo em Negrito utilizando a tag <b> (Não Semantica).

Nessa frase tenho um termo em Destaque utilizando a tag <strong> (Semantico, quer dizere algo de força/destaque).

Itálico / Enfasê

Já nesta frase temos um termo em Italico utilizando a tag <i> (Não Semantico).

Nesta frase temos um termo em enfasê utilizando a tag <em> (Semantica).

Marcação / Grifar

Caso queira marcar um texto em especifico, utlizamos a tag <mark>, podemos mudar a cor usando CSS ou introduzindo o "style" dentro da tag desejada.
Caso digite outro texto marcado, note que ele não vai herdar o comando de cor, para isso insira um <style> no <head> introduzidno um "mark" {}.

Texto Deletado

Ja se quisermos deletar um texto, riscando ele, devemos utilizar a tag <del> (Semantica).

Texto Inserido

Caso queira inserir um texto, que seria o oposto de deletar, usaremos a tag <ins> (Semantica).

Textos Inseridos

Para inserir coisas dos tipo: x20=y e H2O
Utilizamos as tags <sup> e <sub>, sendo sup para superior e sub para inferior.
